Comment exporter vos transactions

📊 Binance

  1. Connectez-vous à Binance
  2. Allez dans Ordres > Historique des trades
  3. Cliquez sur "Exporter"
  4. Sélectionnez la période souhaitée
  5. Téléchargez le fichier CSV

📊 Kucoin

  1. Connectez-vous à Kucoin
  2. Allez dans Ordres > Historique des transactions
  3. Cliquez sur "Exporter"
  4. Choisissez le format CSV
  5. Téléchargez le fichier

📝 Format personnalisé

Votre fichier CSV doit contenir les colonnes suivantes : 

Date,Type,Symbol,Volume,Price,Fee
2024-01-15,BUY,BTC,0.5,40000,10
2024-02-20,SELL,BTC,0.1,45000,5
